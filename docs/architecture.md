# Architecture Overview

Omecord is built around several core systems:

## 1. Matchmaking System
Handles:
- Random VC selection
- Multi‑user room creation
- Session lifecycle

## 2. Video System
Manages:
- Video streaming
- Screensharing
- Video‑based visuals
- NSFW filtering

## 3. Support System
Features:
- Multi‑line support VCs
- Queueing
- Staff availability detection

## 4. Moderation Pipeline
Includes:
- Report button
- 10‑second evidence capture
- Staff review flow

## 5. Bot Core
Handles:
- Commands
- Buttons
- Embeds
- Permissions
- Configuration
